WhatsApp Message API Documentation

API Overview

This API allows users to send WhatsApp messages using a predefined sender profile. Users must authenticate with a username and password.

Endpoint

POST /api/

Request Headers

  • Content-Type: application/json

Request Body (JSON)

{
    "username": "your_username",
    "password": "your_password",
    "sender id" "your_senderID"
    "phone_numbers": ["+1234567890", "+9876543210"],
    "message": "Hello, this is a test message!"
}

Response

Success (200)

{
    "success": "Message sent successfully!"
}

Errors

  • 400 Bad Request: Invalid JSON format or missing parameters
  • 403 Forbidden: Invalid username or password
  • 404 Not Found: Sender profile not found
  • 405 Method Not Allowed: Only POST method is allowed
